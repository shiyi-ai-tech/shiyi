# 文件操作能力说明

## 概述

吏员提供安全受限的文件读写能力，所有操作限制在沙箱目录内。

## 沙箱目录

- **路径**: `~/.shiyi/workspace/`
- **说明**: 所有文件操作必须在此目录内进行
- **安全**: 支持相对路径（相对于沙箱目录）和绝对路径（在沙箱内）

## 工具列表

### file_read

读取文件内容或列出目录。

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| path | string | 是 | 文件路径（沙箱内） |
| limit | integer | 否 | 最大读取行数（默认500） |

**使用示例**:
```json
{
  "tool": "file_read",
  "params": {
    "path": "test.txt",
    "limit": 100
  }
}
```

**返回示例**:
```json
{
  "success": true,
  "result": "文件内容...",
  "error": ""
}
```

### file_write

写入内容到文件。

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| path | string | 是 | 文件路径（沙箱内） |
| content | string | 是 | 要写入的内容 |

**使用示例**:
```json
{
  "tool": "file_write",
  "params": {
    "path": "output.txt",
    "content": "Hello, World!"
  }
}
```

**返回示例**:
```json
{
  "success": true,
  "result": "已写入: /home/user/.shiyi/workspace/output.txt (12 字节)",
  "error": ""
}
```

## 安全机制

1. **路径归一化**: 所有路径经过 resolve() 处理
2. **路径穿越检测**: 检查归一化后的路径是否以沙箱根目录开头
3. **目录保护**: 不允许覆盖目录
4. **自动创建**: 写入时自动创建父目录

## 注意事项

- 文件不存在时返回错误
- 超出 limit 行数时截断并提示
- 读取目录时返回文件列表
- 使用 UTF-8 编码，错误字符自动替换
